Tulip-HFT is a piece of software that is build to automatically trade cryptocurrency on various exchanges. The software exists out of a The trading bot a command line application with a configuration file that can be easily created and modified with our GUI. The bot and GUI are separate programs and can be run on different systems. For example run the bot on Linux and use the GUI on Windows.

Low maintenance

The software only needs a one time configuration from which on it can be run completely headless (no user interface needed). Profit can be monitored with external opensource tools.

Our own technology

Our algorithms are completely custom and are improved every release.

Proven to work

We have been developing and testing our own algorithms for over a year with good results. In 2014 we went from 1 BTC to over 40 for which supportive data is available.

No nonsense

Our user interface is simple and does not have 20 sliders and check boxes that need constant babysitting.

Secure

We do not use or send your API keys to our servers.

Supported exchanges

BTCChina, BTCE, Bter, CEX.IO, Hitbtc, ItBit, Kraken
Support for almost all combinations of currencies and cryptocurrencies (only on CEX we do not).

Supported platforms

  • Windows
  • Linux (Debian, Ubuntu, CentOS upon request)
  • OS X support not planned
  • Raspberry Pi upon request
